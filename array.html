Array

Define um lista de elementos. Cada elemento possui um indice que começa a contar do 0.

pop: exclui um elemento da lista (o ultimo se não for definido)

push: adiciona um elemento ao final da lista

length: tamanho da lista

reverse: reverte a ordem dos itens da lista

toString: transforma o array em string

join: altera o simbolo de separação da lista

Dicionário

Define parametros para um dado elemento e cada paramentro possui um nome.

Lista de Dicionário

Define uma lista com vários elementos aos quais cada um possui parametros.
